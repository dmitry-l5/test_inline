проек состоит из :
migrate.php - сщздает таблицы post и comments. База данных должна дыть солдана и логин/пароль должны быть занесены в файлconfig.php до запуска скрипта.
import.php - выполняет импорт данных из указанных URL в БД.
index.php отображает форму для поиска и резуддьтат поиска
find.php - выполняет поиск по дб и возвращает ответ на AJAX запрос
